项目概述

DesktopPetByAi 是一个智能桌面宠物项目，提供交互式桌面陪伴功能

核心定位

DesktopPetByAi 的核心定位是提供交互式桌面陪伴功能，支持与用户对话交流、协助执行简单操作（如打开软件），并集成了车票查询、地理编码等扩展工具。

项目状态

项目目前处于测试阶段，功能可扩展性较强，用户可通过探索挖掘更多用途。

主要特点

  • 智能对话交流，支持自然语言交互
  • 桌面宠物形态，可自定义动画和外观
  • 文本转语音功能，AI回复可语音播放
  • 集成12306车票查询、地理编码等实用工具
  • 模块化设计，便于功能扩展和定制
  • 提供EXE安装包，一键安装使用

重要注意事项

  • AI对话历史存储在ai_memory/memory_default.json，删除该文件可清除记忆
  • 不可删除ai_memory文件夹本身，否则可能导致程序异常
  • 项目为测试版，可能存在功能缺陷，发现问题请及时反馈
核心功能

功能模块详解

深入了解DesktopPetByAi的各项核心功能模块

交互界面（settingwindow.py）

基于Qt框架的图形化交互界面

主要功能

  • 聊天界面：基于Qt框架的QWidget实现，包含聊天历史显示区和输入区
  • 界面样式：通过CSS样式表定义组件外观，支持用户输入提示
  • 消息区分：用户消息为蓝色，AI消息为红色，清晰区分对话角色

核心逻辑

  • 加载历史对话（从ai_memory读取）并显示
  • 处理发送/清空消息操作，发送时禁用按钮防止重复提交
  • 通过工作线程（AIWorker）异步调用AI接口，避免界面卡顿
  • 接收AI回复后更新聊天记录、触发语音播放，并通知主窗口刷新宠物动画

AI交互模块

对话管理：负责加载/保存对话历史，维护ai_memory中的对话数据
智谱AI接口：对接智谱AI开放平台，发送用户消息并获取AI回复
专用模块：zhipu_12306.py专门处理与车票查询相关的AI指令解析

文本转语音（toVoice.py）

功能实现：使用edge_tts库将文本转换为语音，生成临时MP3文件
语音播放：通过pygame库播放生成的音频文件，播放完成后自动删除临时文件
异常处理：播放出错时尝试删除临时文件，避免资源残留

扩展工具模块

12306车票查询：提供get_tickets方法，查询车票信息并返回文本结果
逆地理编码：通过regeocode方法将经纬度坐标转换为具体地址
外部接口：依赖外部API接口实现地理信息解析等高级功能

安装与使用指南

详细的使用流程与注意事项

下载安装

1

下载安装包

直接下载已构建的EXE安装文件，访问 安装页面 获取下载链接。

2

运行安装程序

双击下载的EXE文件，按照安装向导提示完成安装过程。

3

启动应用

安装完成后，可以在桌面或开始菜单找到应用快捷方式，启动后通过系统托盘配置AI密钥等信息。

源码安装

1. 克隆仓库

$ git clone https://github.com/cjz-wr/DesktopPetByAi.git

2. 进入项目目录

$ cd DesktopPetByAi

3. 安装依赖包

$ pip install -r requirements.txt

4. 运行应用

$ python main.py
